XRK Tools — Public Edition
Welcome to the public-facing repository for the XRK Tool Suite — a collection of lightweight, fast, terminal‑based utilities designed to make schoolwork and everyday computing easier.

This repo contains the Free Edition of XRK tools, along with documentation, screenshots, and the public roadmap.
What This Repo Contains:
Free Edition tools (Windows, Linux/ChromeOS coming soon)

Screenshots and previews

Public documentation

Feature lists

Public changelog

Roadmap for upcoming versions

These tools are bare‑bones, forever‑free, and updated only for critical bug fixes.
What This Repo Does Not Contain
The following items are NOT included here:

Paid Edition tools

Updater Pro

GUI versions(coming soon)

Advanced features

Internal dev builds

Source code

All paid and internal builds are stored in a private repository.   
Free Edition Tools (Current)
These tools are included in the Free Edition:

Quick Notes Free — Simple note‑taking with instant save

File Sorter Free — Basic folder cleanup

System Specs Free — Lightweight system info viewer

Free Edition tools are intentionally limited and may fall behind the paid versions.
